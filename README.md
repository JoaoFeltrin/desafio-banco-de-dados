# Desafio de Banco de Dados

Este repositório contém a solução completa do desafio da disciplina **Fundamentos de Banco de Dados** (UNISINOS).

## 📌 Objetivo
Criar uma base de dados completa e funcional, passando pelas etapas:
- Modelagem conceitual (DER)
- Modelagem lógica (Modelo Relacional)
- Normalização
- Implementação em Oracle SQL Live
- Consultas em Álgebra Relacional (Relax)

## 📂 Estrutura
- `docs/` → Diagramas exportados do BRModelo e relatório final
- `sql/` → Script SQL executável no Oracle SQL Live
- `algebra/` → Consultas em álgebra relacional

## 🛠️ Tecnologias
- BRModelo (modelagem conceitual e lógica)
- Oracle SQL Live (implementação)
- Relax (álgebra relacional)
